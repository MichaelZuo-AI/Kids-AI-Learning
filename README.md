# Kids AI Learning

An interactive AI course for kids (ages 6-12), built with Reveal.js and deployed on GitHub Pages.

**Live site: [kids.michaelzuo.vip](https://kids.michaelzuo.vip)**

## Features

- Bilingual (Chinese/English) toggle on every lesson
- Mewtwo-themed character guide throughout the course
- Interactive activities: quizzes, flip cards, classifiers
- Mobile-friendly — works on phones, tablets, and desktops
- No setup required — just open the link

## Tech Stack

- [Reveal.js](https://revealjs.com/) 5.1 (self-hosted)
- Static HTML + CSS — no build step
- GitHub Pages with custom domain
